Explaination

First, we get the middle element of the entire array which is called “mid”. If mid is even, then we compare the mid value to mid+1. If they are equal, it means that the number appears only once is on the right and needs to be found in the right half. Because when mid is even, the length of the divided array is even. If mid and mid+1 are equal, then the number of elements in the right half is odd except for mid+1, and the element that appears only once must appear in the right half. Similarly, we can handle the case when the mid is odd.
